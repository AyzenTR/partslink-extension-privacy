# Règles de confidentialité - Extension "PartsLink AI Scraper"

Dernière mise à jour : 9 juillet 2024

## Introduction
Cette politique de confidentialité explique comment l'extension "PartsLink AI Scraper" pour Chrome collecte, utilise et protège vos données lors de son utilisation sur partslink24.com pour la recherche automatisée de pièces automobiles.

## Données collectées
Notre extension collecte et stocke les données suivantes :
- Numéros VIN saisis par l'utilisateur
- Noms de pièces recherchées
- Contenu HTML des pages visitées (pour analyse AI)
- Logs d'activité de l'extension
- Données de navigation sur partslink24.com
- Résultats de recherche et pièces trouvées

## Utilisation des données
Les données collectées sont utilisées pour :
- Analyser les pages web avec l'intelligence artificielle locale
- Automatiser la navigation et la recherche de pièces
- Fournir des logs d'activité à l'utilisateur
- Améliorer la précision des recherches
- Stocker les résultats de recherche localement

## Traitement par Intelligence Artificielle
L'extension utilise un modèle AI local (Gemini 2.0 Flash) pour :
- Analyser le contenu HTML des pages
- Prendre des décisions de navigation automatisées
- Identifier les pièces automobiles pertinentes
- Optimiser les stratégies de recherche

**Important** : Toute l'analyse AI est effectuée localement sur votre appareil. Aucune donnée n'est envoyée à des serveurs externes pour l'analyse AI.

## Stockage des données
- Toutes les données sont stockées localement via Chrome Storage API
- Les données HTML sont temporairement stockées pour l'analyse AI puis supprimées
- Les logs d'activité sont conservés localement (maximum 1000 entrées)
- Aucune synchronisation avec des serveurs externes
- Vous pouvez supprimer toutes les données en désinstallant l'extension

## Permissions du navigateur
L'extension demande les permissions suivantes :
- "storage" : pour sauvegarder les données de recherche et logs
- "activeTab" : pour interagir avec la page active
- "scripting" : pour injecter les scripts de scraping
- "tabs" : pour la navigation automatisée
- Accès au site partslink24.com : pour fonctionner sur ce site spécifique

## Sécurité et Confidentialité
- Aucune donnée n'est transmise à des serveurs externes
- L'AI fonctionne entièrement en local
- Les identifiants de connexion ne sont pas stockés de manière permanente
- Le contenu HTML est analysé localement puis supprimé
- Toutes les communications restent dans votre navigateur

## Vos droits
Vous pouvez à tout moment :
- Consulter les données stockées via l'interface de l'extension
- Supprimer vos données en désinstallant l'extension
- Réinstaller l'extension pour recommencer avec un stockage vide
- Arrêter le processus de scraping à tout moment
- Effacer les logs d'activité via l'interface

## Limitations et Responsabilités
- L'extension est conçue uniquement pour une utilisation sur partslink24.com
- L'utilisateur est responsable de respecter les conditions d'utilisation du site
- L'utilisation automatisée doit respecter les politiques du site web
- L'extension ne contourne aucune mesure de sécurité du site
