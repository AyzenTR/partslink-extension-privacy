# Règles de confidentialité - Extension "Panier de Références"

Dernière mise à jour : 28 mai 2025

## Introduction
Cette politique de confidentialité explique comment l'extension "Panier de Références" pour Chrome collecte, utilise et protège vos données lors de son utilisation sur partslink24.com.

## Données collectées
Notre extension collecte et stocke uniquement :
- Les références de pièces que vous sélectionnez
- Les descriptions associées à ces références

## Utilisation des données
Les données collectées sont :
- Stockées localement dans votre navigateur Chrome
- Utilisées uniquement pour créer votre liste personnelle de références
- Non partagées avec des tiers
- Non transmises à des serveurs externes

## Stockage des données
- Toutes les données sont stockées localement via Chrome Storage API
- Les données sont synchronisées uniquement entre vos appareils Chrome connectés
- Vous pouvez supprimer ces données à tout moment en désinstallant l'extension

## Permissions du navigateur
L'extension demande les permissions suivantes :
- "storage" : pour sauvegarder vos références sélectionnées
- "activeTab" : pour ajouter les boutons sur la page active
- Accès au site partslink24.com : pour fonctionner sur ce site spécifique

## Vos droits
Vous pouvez à tout moment :
- Consulter les données stockées
- Supprimer vos données en désinstallant l'extension
- Réinstaller l'extension pour recommencer avec un stockage vide
